全カラム型に対応した基本テーブルです。ソート、フィルター、セル編集、クリップボード操作、キーボード操作、行の並び替えが使えます。

- ダブルクリックでセル編集
- 列ヘッダーのメニューからソート / フィルター
- Shift+クリックで範囲選択
- Ctrl+C / Ctrl+V でコピー / ペースト
- 行ヘッダーをドラッグで並び替え
- 左1列が固定表示 (横スクロールしても追従)
