組み込みバリデーション (required, min/max, pattern) とカスタムバリデーション (高スコア警告) のデモです。

- エラーセルは赤、警告セルはオレンジでハイライト
- セルにホバーするとエラーメッセージがツールチップで表示
- 「元に戻す」/「やり直し」で Undo / Redo
- `cellMeta` によるセルごとのカスタムスタイル・ツールチップ:
  - Category "C" のセルはオレンジでハイライト (ホバーで「要レビュー対象」と表示)
  - Score が 50 未満のセルは青でハイライト (ホバーでスコアを表示)
  - Approved が未承認のセルはホバーで注意メッセージを表示
